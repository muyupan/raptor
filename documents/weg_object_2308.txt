This document describes the Fort-28 Ukrainian 5.7mm Pistol.

The Fort-28 Ukrainian 5.7mm Pistol is a double-action-only polymer-framed semi-auto pistol. It is either striker-fired or has a shrouded hammer just like the FN Five-seveN.

This equipment has proliferated into the following countries: Ukraine.

Here is System information for the Fort-28 Ukrainian 5.7mm Pistol:
The Alternate Designation is Fort-28.
The Type is Semi-Automatic Pistol.
The Caliber is 5.7mm.
The Manufacturer is SIA FORT.
The Action is Semi-Automatic.
The Trigger Type is Double Action.
The Magazine is 20 round magazine.
The Sights is Iron.
The Note is The Fort 28 pistol uses an indigenous delayed blowback system where the barrel is allowed to recoil for a short distance, but there’s no rigid link between the barrel and the slide at the moment of discharge. The pistol uses a striker fired, Double Action Only trigger system with no manual safeties and an automated trigger safety. The frame of the gun is made from polymer covered hard steel..
