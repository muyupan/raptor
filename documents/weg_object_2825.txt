This document describes the NPI-2 Russian Personal Navigation Receiver.

The NPI-2 receiver is a personal navigation device that provides geolocation using the GLONASS satellite constellation. The device is widely integrated with the Russian armed forces, primarily enabling navigation, battle tracking, and situational awareness The Global Navigation Satellite System (GLONASS) is a space based satellite navigation system developed by the former Soviet Union as an alternative to the American Global Positioning System (GPS). It is operated and maintained by the Russian Space Agency “Roscosmos”.

This equipment has proliferated into the following countries: Russian Federation.
