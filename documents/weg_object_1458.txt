This document describes the Chong Jin Class North Korean Patrol Boat.

The CHONG JIN class is an improved version of the CHAHO class, with the same hull and propulsion, but with different armament and surface search radar. Construction of the class began in 1975. By about 1985, this class was replaced in construction by the larger CHONG JU class. At least 52 units of this class were built.

This equipment has proliferated into the following countries: North Korea (Democratic People's Republic of Korea).
